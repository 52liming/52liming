# Binary Search Tree（二分查找树）
BST是二叉搜索树（Binary Search Tree）的缩写。它是一种常用的数据结构，用于在计算机科学中存储和组织数据。在二叉搜索树中，每个节点最多有两个子节点，一个左子节点和一个右子节点。左子节点包含小于父节点的值，而右子节点包含大于父节点的值。这样可以实现对树中节点的高效搜索、插入和删除。BST常用于排序和搜索等算法中。
