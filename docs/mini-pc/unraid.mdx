### Docker 镜像加速

https://docker.mirrors.ustc.edu.cn

https://mirror.baidubce.com 

使用阿里云镜像加速

1. 进入阿里云 https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors 获取镜像加速地址 ```https://xxx.mirror.aliyuncs.com```
2. 进入unraid Terminal后台，输入
```bash
mkdir -p /etc/docker
tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://xxx.mirror.aliyuncs.com"]
}
EOF
```
3. 使每次重启后依然生效 `vi /boot/config/go`
```bash
tee /etc/docker/daemon.json <<-'EOF'
{
"registry-mirrors": ["https://xxx.mirror.aliyuncs.com"]
}
EOF
```
4. 重启docker `/etc/rc.d/rc.docker restart`
